## Start a new F# project
```bash
dotnet new console -lang "F#" -o src/App
```
For more examples or options try these two options:
```bash
dotnet new --help
```
```bash
dotnet new
```
## Run the project
At the folder with `App.fsproj`:
```bash
dotnet run <args>
```
